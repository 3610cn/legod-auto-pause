# ·legod-auto-pause
雷神加速器时常自动暂停

      之前的在网上的脚本发现用不了了，
      所以自己写了一个脚本放在这里。
## · 用前须知

1. 源码只是放上来看看，想要程序去上面**release**里面下载zip（只提供exe文件）

1. 下载后先用记事本打开config.ini配置一下用户名和密码，其他配置项可选，注释很清楚

2. token的优先级大于用户名密码，所以不填账户密码只填token也是可以用的

3. token会过期，如果填入用户名密码会自动获取新的token
